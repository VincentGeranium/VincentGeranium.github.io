# Error Handling

- 스위프트에서 오류는 `Error`라는 `프로토콜을 준수하는 타입의 값을 통해 표현`

- Error 프로토콜은 사실상 요구사항이 없는 빈 프로토콜

- Error 프로토콜은 오류를 표현하기 위한 타입(주로 열거형)

- 열거형은 오류의 종류를 나타내기에 아주 적합한 기능, 연관 값을 통해 오류에 관한 부가 정보를 제공할 수도 있다
